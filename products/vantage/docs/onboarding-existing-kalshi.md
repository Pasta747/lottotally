# Vantage Beta Onboarding — Existing Kalshi User
**For: Beta tester who already has a Kalshi account**

---

## What You'll Get
Vantage is an AI sports betting agent that scans markets 24/7, finds +EV (positive expected value) opportunities, and sizes them to your bankroll using Kelly criterion. During beta, everything runs in **paper trading mode** (no real money).

You'll receive signals via WhatsApp and can approve or pass on each one.

---

## Setup Steps (10 minutes)

### Step 1: Get Your Kalshi API Key

1. Log in to **https://kalshi.com**
2. Go to **Settings → API** (or visit https://kalshi.com/account/api directly)
3. Click **"Create API Key"**
4. Give it a name like "Vantage Beta"
5. **Important:** Set permissions to **Read + Trade**
6. Copy both:
   - **API Key ID** (looks like: `abc123def456`)
   - **API Secret** (shown once — save it!)
7. Store these somewhere safe (password manager recommended)

### Step 2: Set Up Paper Trading Mode

Your Kalshi account has a **demo/paper mode**:
1. Go to https://demo.kalshi.com (or toggle "Demo" in the Kalshi app)
2. Your demo account comes with fake funds for paper trading
3. The API key works on both live and demo — we'll point Vantage at the demo endpoint

**You don't need to deposit any real money for the beta.**

### Step 3: Send Us Your Info

Reply to your Vantage WhatsApp thread (or send to Mario) with:

1. ✅ **Kalshi API Key ID**
2. ✅ **Kalshi API Secret**
3. ✅ **Your WhatsApp number** (for signal delivery)
4. ✅ **Paper bankroll amount** — how much fake money do you want to start with? (default: $1,000)
5. ✅ **Risk level** — conservative, moderate, or aggressive
   - **Conservative:** Half-Kelly sizing (smaller bets, slower growth, less variance)
   - **Moderate:** Full Kelly (optimal mathematical sizing)
   - **Aggressive:** 1.5x Kelly (bigger bets, more variance, faster if winning)

### Step 4: You're In

Once we configure your agent:
- You'll receive a welcome message on WhatsApp
- Signals will start coming as the scanner finds opportunities
- Each signal shows: event, side, EV%, suggested bet size
- Reply **YES** to paper-execute, or **PASS** to skip
- Daily summary arrives each evening with your P&L
